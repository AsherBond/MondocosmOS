IGESToBRep_BasicCurve Class Reference

Provides methods to transfer basic geometric curves entities
from IGES to CASCADE.
These can be :
* Circular arc
* Conic arc
* Spline curve
* BSpline curve
* Line
* Copious data
* Point
* Transformation matrix
