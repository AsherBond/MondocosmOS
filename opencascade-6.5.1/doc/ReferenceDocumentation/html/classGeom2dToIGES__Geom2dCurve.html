Geom2dToIGES_Geom2dCurve Class Reference

This class implements the transfer of the Curve Entity from Geom2d
To IGES. These can be :
Curve
. BoundedCurve
* BSplineCurve
* BezierCurve
* TrimmedCurve
. Conic
* Circle
* Ellipse
* Hyperbloa
* Line
* Parabola
. OffsetCurve
