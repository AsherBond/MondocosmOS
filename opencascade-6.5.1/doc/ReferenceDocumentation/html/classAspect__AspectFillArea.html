Aspect_AspectFillArea Class Reference

Group of attributes for the FACE primitives.
The attributes are:
* type of interior
* type of hatch
* interior colour
* border colour
* type of border
* thickness of border
when the value of the group is modified, all graphic
objects using this group are modified.
