 Creates a Wedge algorithm. <Axes> is the axis
system for the primitive.

XMin, YMin, ZMin are set to 0
XMax, YMax, ZMax are set to dx, dy, dz
Z2Min = ZMin
Z2Max = ZMax
X2Min = XMin
X2Max = XMax
The result is a box
dx,dy,dz should be positive
