Manipulating Geometry using Draw

 

The DrawTrSurf package supports the display of parametric curves and surfaces.

The Drawable deferred class inherits the Drawable3D class from the package Draw, it adds methods to draw 3D Curves and Curves on 3D Surfaces.

The classes Curve Curve2d and Surface inherit Drawable and can be used to draw a single curve from packages Geom or Geom2d or a surface from Geom.

The Triangulation and Polygon from the package Poly are also supported.
