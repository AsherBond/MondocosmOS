Welcome to Open CASCADE Technology 6.5.1

 

Welcome to Open CASCADE Technology version 6.5.1, a maintenance release, which introduces a number of new features and improved traditional functionality along with some changes over the previous minor release 6.5.

This release makes Open CASCADE Technology even a more powerful and stable development platform for 3D modeling and numerical simulation applications.

Open CASCADE Technology 6.5.1 is a full-featured package that allows developing applications on Windows and Linux platforms.
