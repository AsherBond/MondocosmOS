DESCRIPTION

r3.colors.out allows the user to export the color table for a raster3d map to a file which is suitable as input to r3.colors.

EXAMPLES

r3.colors.out map=volume_1 rules=rules.txt
r3.colors map=volume_2 rules=rules.txt

SEE ALSO

r3.colors r.colors
